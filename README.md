# BDO_Visualiser
Code Base:
- Angular 11
- Electron.js
- Sqlite DB
- Zing Charts

## Application
Fan-made project to keep track of various activities in the MMORPG Black Desert Online.

## Aim
Provide visual feedback to data given by user.
 
 ## Topics
 - Combat Stats (PvE)
 - Life Stats
 - Fail Stack Tracker
 - Scroll Tracker
 - Barter Stats
 - User Classes

## Current Stage
App is WIP and not V.1 ready yet.
